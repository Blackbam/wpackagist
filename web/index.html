WordPress Packagist

Manage your plugins with Composer

This site mirrors the WordPress plugin Subversion repository as a Composer repository.

Here's an example composer.json file which adds the Wpackagist repository and includes the latest version of Advanced Custom Fields and version 1.4 of Posts to Posts:

"name": "acme/brilliant-wordpress-site",
"description": "My brilliant WordPress site",
"repositories":[
    {
        "type":"composer",
        "url":"http://wpackagist.org"
    }
],
"require": {
    "acme/example-lib":"*",
    "wpackagist/advanced-custom-fields":"*",
    "wpackagist/posts-to-posts":"1.4.x"
},
"autoload": {
    "psr-0": {
        "Acme": "src/"
    }
}
